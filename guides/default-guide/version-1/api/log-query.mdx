Parseable log query API allows you to query logs in real-time. You can query logs using standard PostgreSQL syntax. The API needs `startTime` and `endTime` parameters to query logs. 

[![Run in Postman](https://run.pstmn.io/button.svg)](https://god.gw.postman.com/run-collection/22353706-8555bcd5-8af3-4fe9-9e55-52867217561c?action=collection%2Ffork&collection-url=entityId%3D22353706-8555bcd5-8af3-4fe9-9e55-52867217561c%26entityType%3Dcollection%26workspaceId%3De84052ac-c01a-435a-bb68-d2076623474a)

[Log Stream Query API](https://www.postman.com/parseable/workspace/parseable/request/22353706-7b281b33-2f37-4034-9386-9a78e37b1db1)

:::tip Time zone and format

Parseable expects both the start and end time in UTC, in RFC 3339 and ISO 8601 date and time string such as `2022-09-10T05:00:00+00:00` format.

:::
