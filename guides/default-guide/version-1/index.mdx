Parseable is a cloud native log data analytics engine. Parseable is written in Rust and uses Apache Arrow and Parquet as underlying data structures.

- Deploy anywhere - public or private cloud, containers, VMs, or bare metal environments.
- Up to 5x smaller memory and compute footprint compared to JVM-based solutions.
- Single binary with built-in UI, setup within minutes.
- Ingests logs, metrics, and traces in OpenTelemetry format.
- Supports structured and unstructured data.
- Real-time and historical data analytics.
