## viewer

Lookup the details of the authenticated user.

### Example Request

```graphql
query {
  viewer {
    id
    firstName
    lastName
    email
  }
}
```

### Example Response

```graphql
{
  "data": {
    "viewer": {
      "id": "VXNlcjp6a2dkZXk=",
      "firstName": "John",
      "lastName": "Doe",
      "email": "john.doe@email.com",
    }
  }
}
```
