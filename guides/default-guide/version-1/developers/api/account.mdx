The `Account` query is used to fetch the details of your team or individual account on DeepSource. It also provides mutations to configure account-wide settings.

## account

`Query:account` Fetch the details of a single DeepSource account.

### Arguments

<table>
  <thead>
    <tr>
      <th width="25%">Field</th>
      <th width="25%">Type</th>
      <th width="50%">Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>login</td>
      <td><a href="https://deepsource-docs.readme.io/docs/apis-reference-scalars#string">String</a>!</td>
      <td>
          The login or username to lookup the account by.
      </td>
    </tr>
    <tr>
      <td>vcsProvider</td>
      <td><a href="https://deepsource-docs.readme.io/docs/apis-reference-enums#vcsprovider">VCSProvider</a>!</td>

### Sample Request

```graphql
query {
  account(login: "deepsourcelabs", vcsProvider: GITHUB) {
    login
    id
    type
    vcsProvider
    vcsUrl
    avatarUrl
  }
}
```

### Sample Response

```json
{
  "data": {
    "account": {
      "login": "cyberdyne",
      "id": "QWNjb3VudDp6bWpws2I=",
      "type": "TEAM",
      "vcsProvider": "GITHUB",
      "vcsUrl": "https://github.com/cyberdyne",
      "avatarUrl": "https://user-avatars.deepsource.io/demo-image.png"
    }
  }
}
```
