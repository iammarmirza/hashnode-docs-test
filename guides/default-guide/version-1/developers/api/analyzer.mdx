## analyzer

`Query: analyzer` Fetch details of a single analyzer using the shortcode.

### Arguments

<table>
  <thead>
    <tr>
      <th width="25%">Field</th>
      <th width="25%">Type</th>
      <th width="50%">Description</th>
    </tr>
  </thead>
  <tbody>
<tr>
<td align="left" valign="top">shortcode</td>
<td valign="top"><a href="https://deepsource-docs.readme.io/docs/apis-reference-scalars#string">String</a>!</td>
<td>Shortcode of the analyzer you'd like to get.</td>
</tr>
</tbody>
</table>

### Sample Request

```graphql
query {
  analyzer(shortcode:"python") {
    id
    shortcode
    name
  }
}
```

### Sample Response

```json
{
  "data": {
    "analyzer": {
      "id": "QW5hbHl6Zdd6bGtiZXZ6",
      "shortcode": "python",
      "name": "Python"
    }
  }
}
```

```graphql
query {
  analyzers{
   	edges {
      node {
        id
        name
        shortcode
      }
    }
  }
}
```

```json
{
  "data": {
    "analyzers": {
      "edges": [
        {
          "node": {
            "id": "QW5hbHl6ZXI6bGtiZXZ6",
            "name": "Python",
            "shortcode": "python"
          }
        },
        {
          "node": {
            "id": "QW5hbHl6ZXI6cmx6b2xi",
            "name": "Go",
            "shortcode": "go"
          }
        }
      ]
    }
  }
}
```
