## transformer

`Query: transformer` Fetch details of a transformer using the transformer shortcode.

### Arguments

<table>
  <thead>
    <tr>
      <th width="25%">Field</th>
      <th width="25%">Type</th>
      <th width="50%">Description</th>
    </tr>
  </thead>
  <tbody>
<tr>
<td align="left" valign="top">shortcode</td>
<td valign="top"><a href="https://deepsource-docs.readme.io/docs/apis-reference-scalars#string">String</a>!</td>
<td>Shortcode of the transformer.</td>
</tr>
</tbody>
</table>

### Sample Request

```graphql
query {
  transformer(shortcode:"gofmt"){
   	id
    name
    shortcode
  }
}
```

### Sample Response

```json
{
  "data": {
    "transformer": {
      "id": "VHJhbnNmb3JtZXI6bGp6a3d6",
      "name": "Go fmt",
      "shortcode": "gofmt"
    }
  }
}
```

```graphql

  transformers{
   	edges {

        id

        shortcode
```

```json
    "transformers": {
```
