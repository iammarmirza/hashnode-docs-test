This section covers configuration specific to the **Secrets** analyzer. Please make sure you have read the general [configuration guide](https://deepsource-docs.readme.io/docs/configuration) first.

## Configuration - `.deepsource.toml`

### `name`

- **Type**: [String](https://toml.io/en/v1.0.0#string)
- **Presence**: mandatory
- **Description**: Shortcode of the analyzer.
- **Example**:

```toml
name = "secrets"
```

### `enabled`

- **Type**: [Boolean](https://toml.io/en/v1.0.0#boolean)
- **Presence**: optional
- **Description**: Toggle whether this analyzer should be run.
- **Example**:

```toml
enabled = true
```

