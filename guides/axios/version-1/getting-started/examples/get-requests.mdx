`GET` requests are used to retrieve data from a server. Axios makes it simple to send `GET` requests by providing a promise-based interface that works well with both the browser and Node.js.

## Basic `GET` Request

To send a basic `GET` request with Axios, use the `axios.get()` method and pass the URL of the API endpoint you want to request data from.

```javascript
import axios from 'axios';

axios.get('https://api.example.com/data')
  .then(response => {
    console.log('Response data:', response.data);
  })
  .catch(error => {
    console.error('Error fetching data:', error);
  });
```
In this example, Axios fetches data from the given URL and logs the response data to the console. If an error occurs (such as a network issue or a 404), the error is caught and logged.

## `GET` Request with Query Parameters
Sometimes, you need to pass query parameters in a `GET` request. Axios makes it easy to do so by appending the parameters as part of the URL or by using the params option.

```js
axios.get('https://api.example.com/search', {
  params: {
    query: 'axios',
    page: 2,
    limit: 10
  }
})
.then(response => {
  console.log('Search results:', response.data);
})
.catch(error => {
  console.error('Error in search request:', error);
});
```
In this example, Axios automatically appends the query parameters to the URL (?query=axios&page=2&limit=10) and sends the request.

## `GET` Request with Headers
Sometimes, you need to include custom headers in your `GET` request, such as an authentication token. This can be done using the headers option in Axios.

```js
axios.get('https://api.example.com/user', {
  headers: {
    Authorization: 'Bearer your_token_here'
  }
})
.then(response => {
  console.log('User data:', response.data);
})
.catch(error => {
  console.error('Error fetching user data:', error);
});
```
In this case, Axios sends an authorization token as part of the request headers to authenticate the request.

## `GET` Request with `Async/Await`
Using `async/await` syntax makes handling promises easier and makes the code look more synchronous, which improves readability.

```js
const fetchData = async () => {
  try {
    const response = await axios.get('https://api.example.com/data');
    console.log('Data fetched:', response.data);
  } catch (error) {
    console.error('Error fetching data:', error);
  }
};

fetchData();
```
In this example, await is used to wait for the `GET` request to complete, and `try/catch` is used to handle any potential errors.
