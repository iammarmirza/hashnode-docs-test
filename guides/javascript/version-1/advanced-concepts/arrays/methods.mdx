JavaScript arrays come equipped with a wide range of built-in methods that allow for powerful and flexible data manipulation. Below are some commonly used array methods, along with their descriptions and examples.

1. **`push()`**: 
Adds one or more elements to the end of an array and returns the new length of the array.
```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(fruits); // Output: ["apple", "banana", "orange"]
console.log(newLength); // Output: 3
```

2. **`pop()`**: 
Removes the last element from an array and returns that element. This method changes the length of the array.

```js
const fruits = ["apple", "banana", "orange"];
const lastFruit = fruits.pop();
console.log(fruits); // Output: ["apple", "banana"]
console.log(lastFruit); // Output: orange
```

3. **`shift()`**: 
Removes the first element from an array and returns that element. This method changes the length of the array.

```js
const fruits = ["apple", "banana", "orange"];
const firstFruit = fruits.shift();
console.log(fruits); // Output: ["banana", "orange"]
console.log(firstFruit); // Output: apple
```

4. **`unshift()`**: 
Adds one or more elements to the beginning of an array and returns the new length of the array.

```js
const fruits = ["banana", "orange"];
const newLength = fruits.unshift("apple");
console.log(fruits); // Output: ["apple", "banana", "orange"]
console.log(newLength); // Output: 3
```

5. **`splice()`**: 
Changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

```js
const fruits = ["apple", "banana", "orange"];
fruits.splice(1, 1, "kiwi", "mango"); // Removes 1 element at index 1 and adds "kiwi" and "mango"
console.log(fruits); // Output: ["apple", "kiwi", "mango", "orange"]
```

6. **`slice()`**: 
Returns a shallow copy of a portion of an array into a new array object selected from start to end (end not included).

```js
const fruits = ["apple", "banana", "orange", "kiwi"];
const citrus = fruits.slice(1, 3); // Gets elements from index 1 to 2
console.log(citrus); // Output: ["banana", "orange"]
```

7. **`forEach()`**: 
Executes a provided function once for each array element.

```js
const fruits = ["apple", "banana", "orange"];
fruits.forEach(function(fruit) {
    console.log(fruit);
});
// Output:
// apple
// banana
// orange
```

8. **`map()`**: 
Creates a new array populated with the results of calling a provided function on every element in the calling array.

```js
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(function(num) {
    return num * 2;
});
console.log(doubled); // Output: [2, 4, 6, 8]
```

9. **`filter()`**: 
Creates a new array with all elements that pass the test implemented by the provided function.

```js
const numbers = [1, 2, 3, 4, 5, 6];
const evenNumbers = numbers.filter(function(num) {
    return num % 2 === 0;
});
console.log(evenNumbers); // Output: [2, 4, 6]
```

10. **`reduce()`**: 
Executes a reducer function (that you provide) on each element of the array, resulting in a single output value.

```js
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce(function(total, num) {
    return total + num;
}, 0);
console.log(sum); // Output: 10
```

11. **`find()`**: 
Returns the value of the first element in the provided array that satisfies the provided testing function. Otherwise, it returns undefined.

```js
const numbers = [4, 9, 16, 25];
const firstLargeNumber = numbers.find(function(num) {
    return num > 10;
});
console.log(firstLargeNumber); // Output: 16
```

12. **`includes()`**: 
Determines whether an array includes a certain value among its entries, returning true or false as appropriate.

```js
const fruits = ["apple", "banana", "orange"];
console.log(fruits.includes("banana")); // Output: true
console.log(fruits.includes("grape")); // Output: false
```

13. **`join()`**: 
Joins all elements of an array into a string. You can specify a separator; the default is a comma.

```js
const fruits = ["apple", "banana", "orange"];
const fruitString = fruits.join(", ");
console.log(fruitString); // Output: "apple, banana, orange"
```

14. **`reverse()`**: 
Reverses the elements of an array in place.

```js
const numbers = [1, 2, 3, 4];
numbers.reverse();
console.log(numbers); // Output: [4, 3, 2, 1]
```

15. **`sort()`**: 
Sorts the elements of an array in place and returns the sorted array. By default, the sort method sorts the elements as strings.

```js
const numbers = [4, 1, 3, 2];
numbers.sort(); // Sorts as strings
console.log(numbers); // Output: [1, 2, 3, 4]

const sortedNumbers = [4, 1, 3, 2];
sortedNumbers.sort((a, b) => a - b); // Sorts numerically
console.log(sortedNumbers); // Output: [1, 2, 3, 4]
```

16. **`concat()`**: 
Merges two or more arrays and returns a new array.

```js
const fruits = ["apple", "banana"];
const vegetables = ["carrot", "pea"];
const food = fruits.concat(vegetables);
console.log(food); // Output: ["apple", "banana", "carrot", "pea"]
```
