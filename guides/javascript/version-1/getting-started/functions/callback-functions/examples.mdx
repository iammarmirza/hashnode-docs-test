### 1. Event Handling: 
Callback functions are widely used in event handling. For example, you can define a function to be executed when a button is clicked:

```js
document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});
```
In this example, the anonymous function is passed as a callback to the addEventListener method, which gets triggered when the button is clicked.

### 2.Array Methods: 
JavaScript provides built-in methods like map, filter, and reduce that use callbacks to process array elements.

Example with `map`:
```js
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(function(num) {
    return num * 2;
});
console.log(doubled); // Output: [2, 4, 6, 8]
```

Example with `filter`:
```js
const numbers = [1, 2, 3, 4, 5, 6];
const evenNumbers = numbers.filter(function(num) {
    return num % 2 === 0;
});
console.log(evenNumbers); // Output: [2, 4, 6]
```

Example with `reduce`:
```js
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce(function(total, num) {
    return total + num;
}, 0);
console.log(sum); // Output: 10
```

### 3. Asynchronous Operations: 
Callbacks are extensively used in asynchronous code, especially with timers, setTimeout, and network requests like fetch.

Example with setTimeout:
```js
setTimeout(function() {
    console.log("This message is shown after 2 seconds");
}, 2000);
```
