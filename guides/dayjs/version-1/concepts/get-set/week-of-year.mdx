Gets or sets the week of the year.

> This requires the **`WeekOfYear`** plugin to work.

```js
dayjs.extend(weekOfYear)

dayjs('2018-06-27').week() // 26
dayjs('2018-06-27').week(5) // returns new dayjs object
```
