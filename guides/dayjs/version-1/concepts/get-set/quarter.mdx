Gets or sets the quarter.

<Callout type="info">
This requires the **`QuarterOfYear`** plugin to work.
</Callout>

```js
dayjs.extend(quarterOfYear)

dayjs('2010-04-01').quarter() // 2
dayjs('2010-04-01').quarter(2) // returns new dayjs object
```
