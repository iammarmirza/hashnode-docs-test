In this article, you will learn how to enter/exit the Comment Mode. This mode enables you to manage comments.

1. Find the **Superflow Toolbar** and click the **Comment** icon to access the comment mode.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733292373594/98b92dSaV.png?auto=format)
 

2. You will enter the **Comment Mode** as the browser indicates. When you enter the **Comment Mode**, you will be able to do the following:
* Comment on Elements
* Comment on Text
* Comment on Images
* Draw a Box Comment

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733292390736/G8sJFeTbF.png?auto=format)
 

3. To exit the **Comment Mode**, press the **ESC** on your keyboard or click the **esc to exit** indicator on top of the browser.
