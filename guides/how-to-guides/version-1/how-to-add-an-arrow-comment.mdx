## How to Add an Arrow Comment

In this article, you will learn how to add an arrow comment using the Superflow Toolbar.

1. Find the **Superflow Toolbar** and click the **Arrows** icon.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733291979469/FBUyYq0e0.png?auto=format)
 

2. Click the **Element** or **Text** where you want to point the arrow.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733291993900/3o0o--N7n.png?auto=format)
 

3. The **Arrow** will appear on the selected area. You can move the arrow by clicking and dragging the arrow itself.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733292009490/eFcM8XKF1.png?auto=format)
 

4. You can adjust the arrow size by clicking the end point of the arrow and dragging it to achieve the preferred size.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733292025298/qhVNGrtbE.png?auto=format)
