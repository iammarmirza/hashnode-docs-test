## How to Use Guest Mode

To use the Guest Mode feature, you have to sign out from your Superflow account. Once you log out, the GUEST MODE will automatically kick in. When the guest mode is enabled, you can leave comments on the project even if you are not logged in to a Superflow account.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733293502044/SDjGhIMGD.png?auto=format)
 

It is important to note, that to use the Guest Mode, it has to be enabled first. See Enabling Guest Mode (Admin) for more information.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733293517173/e-BEGAp5i.png?auto=format)
 

### Enabling Guest Mode (Admin)

Enabling the guest mode is a required prerequisite to use this feature. You need to have admin credentials to access this option.

1. Find the **Superflow Toolbar** and click the **More Options** icon.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733293537609/uZQ1gMedr.png?auto=format)
 

2. A sub-menu will expand, click the **View Project** option to select.

3. You will be directed to the **Project Dashboard** page, find and click the **Settings** icon.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733293557934/sKUrUh2un.png?auto=format)
 

4. Find the **Manage Access & Permissions** section and click the **See More** drop-down arrow.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733293569885/0KS1flGpZ.png?auto=format)
 

5. The **Manage Access & Permissions** section will expand, go to the **Guest Mode** option and ensure that it is turned on by toggling it to **Enable Guest Mode**.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733293586769/HpY3qncX2.png?auto=format)
