## How to View the Project Dashboard

In this article, you will learn how to access the Project Dashboard through the Superflow Toolbar. The Project Dashboard requires admin-level credentials as this page enables you to manage projects.

1. Find the **Superflow Toolbar** and click the **More Options** icon.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733293665232/CTKx6IfYG.png?auto=format)
 

2. A sub-menu will expand, click the **View Project** option to select.

3. A new tab will direct you to the **Project Dashboard** page.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733293682762/W-Oh3bdlU.png?auto=format)
 

4. On this page, you will find the following features:

* Project Details (e.g. project URL, People, Pages, Views)
* Manage Versions
* Settings
* Dashboard for Comment Status (e.g. Open, In Progress, Resolved)

5. You can also access the following modules:

* Website
* Files
* Users
* Integrations
* Help Videos
* Contact Support
* Join Community
