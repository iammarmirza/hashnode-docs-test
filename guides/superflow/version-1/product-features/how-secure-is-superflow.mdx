## How Secure is Superflow

Superflow aims to be the most secure annotation and collaboration tool.

Superflow supports multi-tenant architecture, assigns dedicated databases for data isolation, and encrypts data in transit and at rest using industry standards.

We are in the process of becoming SOC II compliant.
