## How to Use Webhooks

In this article, you will learn how to integrate Superflow with Webhooks.

Access the Superflow Dashboard by going to [app.usesuperflow.com](app.usesuperflow.com). Alternatively, when you are in Review Mode, you can use the method below:

1. Find the **Superflow Toolbar** and click the **More Options** icon.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733131835055/7wB0vNvxr.png?auto=format)
 

2. A sub-menu will expand, click the **View Project** option to select.

3. A new tab will direct you to the **Project Dashboard** page.

> **Note**: By default, you will be directed to the Dashboard for Comment Status screen when accessing the **Project Dashboard**.

4. Click the **INTEGRATIONS** module to access more integration options.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733137141492/bTfuV3K7f.png?auto=format)
 

5. Select **WEBHOOKS** to access Webhooksâ€™ integration page.

6. Click the **Enable Webhooks Toggle** bar to enable Webhooks services.

7. After enabling the Webhooks, a **Confirmation** message will appear that the Webhook service is enabled.

 
![](https://cdn.hashnode.com/res/hashnode/image/upload/v1733137158697/aB0kWn2Jl.png?auto=format)
